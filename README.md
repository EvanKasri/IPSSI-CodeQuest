# IPSSI CodeQuest

Plateforme d'apprentissage interactive pour le code, développée pour l'IPSSI Nice.

## Technologies

- React + Vite
- TailwindCSS
- Framer Motion
- CodeMirror
- React Router

## Installation

```bash
npm install
npm run dev
```

L'application sera accessible sur `http://localhost:5173`

## Fonctionnalités

- Cours interactifs (HTML, CSS, JavaScript, Python)
- Éditeur de code intégré
- Validation automatique des exercices
- Prévisualisation en temps réel pour HTML/CSS/JS
- Système d'onglets pour CSS et JavaScript
- Interface responsive

## Structure

```
src/
├── components/     # Composants réutilisables
├── pages/          # Pages de l'application
├── data/           # Données des cours et exercices
└── styles/         # Styles globaux
```

## Cours disponibles

- **HTML** : Structure et balises de base
- **CSS** : Style et mise en page
- **JavaScript** : Programmation interactive
- **Python** : Fondamentaux de la programmation
